# SecureGuard

SecureGuard is a comprehensive security and information report system designed to enhance safety, protect sensitive data, and streamline incident reporting within the University of Benin community.

## Project Aims

### 1. Overview

SecureGuard provides a centralized platform for reporting and managing security incidents, academic issues, and other concerns within the University of Benin. It aims to create a secure digital environment, safeguard academic information, and foster a proactive approach to security.

### 2. Key Features

- **Incident Reporting:** Easily report security incidents, academic issues, or other concerns through a user-friendly interface.
- **Real-time Threat Alerts:** Stay informed about potential risks within the University of Benin community with our real-time threat detection system.
- **User Access Management:** Control and monitor access to academic resources, ensuring the confidentiality of sensitive information.

### 3. Target Audience

SecureGuard is specifically tailored for students, faculty, and staff within the University of Benin. It addresses the unique security challenges and academic needs of the university community.

### 4. Technologies Used

- **Frontend:** Next.js
- **Database:** Firebase

## Getting Started

Follow these instructions to set up and run SecureGuard on your local machine.

### Prerequisites

- Node.js installed
- Git

### Installation

1. Clone the repository
2. Navigate to the project directory: `cd secureguard`
3. Install dependencies: `npm install`
4. Set up environment variables (refer to .env.example)
5. Run the application: `npm un dev`
